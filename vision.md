# VISION — EGB OCCITANIE

Créer le site de référence du BTP premium en Occitanie.

Un site qui :
- inspire confiance en <3 secondes
- élimine la comparaison prix
- positionne EGB comme un acteur évident, pas optionnel

Le site n’a PAS pour objectif :
- de tout expliquer
- de montrer toutes les prestations
- de plaire à tout le monde

Il a pour objectif :
- de rassurer le cerveau reptilien
- de séduire le cortex visuel
- de guider sans effort vers un contact qualifié

Principe fondamental :
> Si le site paraît cher, lent, confus ou “fait par une agence”, il est raté.
