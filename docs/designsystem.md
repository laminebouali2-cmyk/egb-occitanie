# DESIGN SYSTEM — NEUROSCIENCE FIRST

## Charge cognitive
Objectif : < 3 éléments simultanés par écran

- Pas de carrousel
- Pas de texte long
- Pas d’icônes décoratives

Chaque section doit répondre à UNE question mentale.

## Typographie
- Sans serif moderne
- Interlettrage aéré
- Hiérarchie ultra claire

Le texte ne doit jamais “demander un effort”.

## Couleurs
- Palette courte (2–3 max)
- Dominance neutre (blanc / gris / pierre)
- Accent rare → utilisé comme signal neurologique

## Rythme
- Sections aérées
- Blanc = respiration = luxe
- Scroll fluide, sans surprise

## Images / Vidéo
- Ultra réalistes
- Plans larges
- Pas de “fake obvious”

Règle :
> Une image doit suffire à comprendre le niveau de l’entreprise sans lire une ligne.
