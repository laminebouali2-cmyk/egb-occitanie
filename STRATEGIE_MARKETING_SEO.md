- **Longueur :** 1500-2500 mots (SEO-friendly)

## 6. Stratégies SEO Avancées

### 6.1 Maillage interne stratégique
**Règles :**
- Chaque page de quartier → Service principal
- Articles blog → Landing page service + projets
- Homepage → Top 5 pages stratégiques
- Silos thématiques (Construction / Rénovation / Extension)

**Exemple de maillage optimal :**
```
Article "Prix construction Toulouse"
  ↓
  └─ Liens vers :
     ├─ /services/construction-neuve
     ├─ /construction-villa-toulouse
     ├─ /construction-colomiers
     ├─ /construction-balma
     └─ /projets (filtre construction)
```

### 6.2 Schema Markup (Rich Snippets)
**À implémenter :**
- LocalBusiness (toutes pages)
- Service (pages services)
- FAQPage (FAQ + articles)
- Review / AggregateRating (témoignages)
- BreadcrumbList (navigation)
- HowTo (guides / processus)
- VideoObject (si vidéos)

### 6.3 Optimisation technique
**Performance :**
- [ ] Images WebP + lazy loading
- [ ] Core Web Vitals (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- [ ] CDN pour assets statiques
- [ ] Minification CSS/JS
- [ ] Cache agressif

**Mobile-First :**
- [ ] Design responsive (priorité mobile)
- [ ] Bouton appel en sticky sur mobile
- [ ] Formulaires simplifiés mobile

**Crawlabilité :**
- [ ] Sitemap XML à jour
- [ ] Robots.txt optimisé
- [ ] Pas de contenu dupliqué (canonical tags)
- [ ] URLs propres et descriptives

## 8. Tracking & Analytics

### 8.1 KPIs à suivre (Dashboard mensuel)
**SEO :**
- Trafic organique global
- Positions mots-clés stratégiques (Top 20)
- Impressions / CTR (Google Search Console)
- Backlinks (nombre + qualité)
- DA/PA évolution

**Conversion :**
- Taux de conversion site (objectif : 2-3%)
- Nombre leads/mois (objectif : 30-50)
- Coût par lead (organique vs payant)
- Taux transformation lead → client (objectif : 15-20%)

**Engagement :**
- Pages vues/session
- Taux de rebond
- Temps moyen session
- Pages les plus performantes

**Local :**
- Impressions Google Business Profile
- Clics "Appeler" / "Itinéraire" / "Site Web"
- Nombre avis + note moyenne
- Photos vues

**Réseaux sociaux :**
- Followers croissance
- Engagement rate
- Reach posts
- Trafic site depuis social media

### 8.2 Outils nécessaires
- **Google Analytics 4** (tracking conversions)
- **Google Search Console** (SEO monitoring)
- **Google Tag Manager** (événements)
- **Hotjar / Microsoft Clarity** (heatmaps + recordings)
- **SEMrush / Ahrefs** (suivi positions + concurrence)
- **Call tracking** (attribution calls)
